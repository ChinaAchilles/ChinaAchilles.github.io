创建一个新的博客$ hexo new post “name”
编辑完成$ hexo generate和$ hexo deploy就可以成功发表了。
hexo deploy命令是将我们的博客文章发表到我们的GitHub上的Hexo博客，并不是blog仓库
发表完文章，我们还要吧hexo程序同步到GitHub的blog仓库执行命令$ git add .
这是可以输入$ git status查看状态，会显示刚才编辑过的文件的信息
之后输入下面指令完成上传：
$ git comit -m
$ git push -u origin master
